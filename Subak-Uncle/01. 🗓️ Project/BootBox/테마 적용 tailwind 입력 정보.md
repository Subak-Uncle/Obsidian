### 1. 텍스트 색상:
```
기본: text-text
밝은 색: text-text-light
어두운 색: text-text-dark
```

### 2. 테마 색상:
```
주 색상: bg-primary 또는 text-primary
보조 색상: bg-secondary 또는 text-secondary
본문 배경색: bg-body
테두리 색상: border-border
밝은 테마 색상: bg-theme-light
어두운 테마 색상: bg-theme-dark
오버레이 색상: bg-overlay
```

### 3. 다크모드 색상 (다크모드가 활성화된 경우):
```
텍스트 색상: dark:text-darkmode-text
밝은 텍스트 색상: dark:text-darkmode-text-light
어두운 텍스트 색상: dark:text-darkmode-text-dark
주 색상: dark:bg-darkmode-primary 또는 dark:text-darkmode-primary
보조 색상: dark:bg-darkmode-secondary 또는 dark:text-darkmode-secondary
본문 배경색: dark:bg-darkmode-body
테두리 색상: dark:border-darkmode-border
밝은 테마 색상: dark:bg-darkmode-theme-light
어두운 테마 색상: dark:bg-darkmode-theme-dark
오버레이 색상: dark:bg-darkmode-overlay
```
